# Curso Excel — Nível Básico

Repositório público com a descrição e os materiais do **Curso de Microsoft Excel em Nível Básico**.

---

## 📋 Sobre o curso

Este curso foi desenvolvido para quem está começando a usar o Microsoft Excel.
Ao longo das aulas serão abordados os conceitos e recursos fundamentais da ferramenta, como:

- Interface e navegação na planilha
- Inserção e formatação de dados
- Fórmulas e funções básicas (SOMA, MÉDIA, MÍNIMO, MÁXIMO, etc.)
- Criação e formatação de tabelas
- Gráficos simples
- Impressão e exportação de planilhas

---

## ⬇️ Como baixar os materiais

Não é necessário ter uma conta no GitHub para acessar ou baixar os materiais deste repositório.

### Opção 1 — Baixar tudo de uma vez (arquivo ZIP)

1. Acesse a página principal do repositório:
   [https://github.com/hugoventurini/Curso_Excel_N-vel_B-sico](https://github.com/hugoventurini/Curso_Excel_N-vel_B-sico)
2. Clique no botão verde **`<> Code`**
3. Selecione **`Download ZIP`**
4. Salve e descompacte o arquivo no seu computador

### Opção 2 — Baixar um arquivo individual

1. Navegue até o arquivo desejado dentro do repositório
2. Clique sobre o nome do arquivo para abri-lo
3. Clique no botão **`Download`** (ou pressione o ícone de download) para salvá-lo diretamente
